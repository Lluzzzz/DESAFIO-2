Aproximación de Stirling para factoriales grandes

La aproximación de Stirling es una fórmula que estima el valor de factoriales grandes de forma más eficiente que el cálculo directo. Se basa en la fórmula:

n! ≈ √(2πn) * (n/e)^n

BREVE EXPLICACIÓN DEL TEMA

QUÉ ES JAVASCRIPT

JavaScript es un poderoso lenguaje de programación que permite crear funcionalidades dinámicas en páginas web. JavaScript es fundamental para hacer que las páginas web sean más dinámicas y atractivas para los usuarios.

document.write

La función document.write() se utiliza para escribir contenido en el documento HTML en tiempo de carga.

PROMPT

La función prompt() muestra un cuadro de diálogo con un mensaje y permite al usuario ingresar información.

VARIABLES

aSe utilizan para almacenar datos que pueden cambiar durante la ejecución del programa. En JavaScript, puedes declarar variables con var, let y const. var y let permiten reasignar valores, mientras que const crea una variable de solo lectura cuyo valor no puede cambiar una vez asignado.

CONSTANTES

Se utilizan para almacenar valores que no cambian durante la ejecución del programa. Las constantes se declaran con la palabra clave const y su valor no se puede reasignar una vez inicializado.

OPERACIONES QUE PODEMOS HACER

ARITMETICAS

  • Suma: +
  • Resta: -
  • Multiplicación: *
  • División: /
  • Módulo (resto de la división): %
  • Incremento: ++
  • Decremento: --

DE COMPARACIÓN

  • Igualdad: == o ===
  • Desigualdad: != o !==
  • Mayor que: >
  • Menor que: <
  • Mayor o igual que: >=
  • Menor o igual que: <=

Concatenación: +

LÓGICAS

  • AND lógico: &&
  • OR lógico: ||
  • NOT lógico: !
